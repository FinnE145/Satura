The objective

Own the board.

Every cell holds a paint value for each player — zero to five. Whoever painted a cell more owns it. Dominate sixty percent of the board and you win. Let the clock run out and you lose.

The mechanic

Your move
is a script.

Paint the board by writing code that controls your agent. Every script is a bet on what your opponent will do — write robust code to adapt, or efficient code to save words.
